What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications.

Features of HTML

HTML elements are used to build the structure of a webpage.

It defines the skeleton of a webpage.

Defines structure of web pages using elements like headings, paragraphs, and lists.

Supports multimedia like images, videos, and audio.

Creates links between pages

HTML Basic structure

<!DOCTYPE html>

<html>

<head>

<title>

</head>

<body>

!-- Your content goes here--!

</body>

</html>
